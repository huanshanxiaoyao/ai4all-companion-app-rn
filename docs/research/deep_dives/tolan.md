# 产品深拆：Tolan

> 产品ID：PRD-005  
> 状态：Wave 2 官方证据首轮完成；账号内实测待移动端  
> 平台/地区：iOS、Android；全球公开页面  
> 最近核验：2026-07-17

## 1. 产品承诺与关系路线

Tolan 把关系对象设为明确的外星朋友，并强调“不假装成人类”。官方承诺其会倾听、回应、记住用户，并在合适时后退。其研究价值不是更强拟人，而是通过非人身份、视觉形象与关系边界建立温暖感。（官方自述，EVD-008、EVD-135）

## 2. 当前可确认机制

| 机制 | 当前证据 | 性质/限制 |
| --- | --- | --- |
| 记忆 | 应用内对话会被记录并用于生成 Tolan 的记忆 | 官方自述，EVD-135 |
| 隐私与退出 | 支持在应用内删除账号和相关数据 | 官方自述，EVD-135 |
| 年龄 | 面向 16 岁及以上用户 | 官方自述，EVD-135 |
| 商业模式 | Basic 提供有限免费访问，完整权益与价格待商店内核验 | 官方自述，EVD-135 |
| 规模 | Android 公开安装区间仍低于本轮 100K 门槛，因模式独特保留为标杆 | Google Play，EVD-017 |

## 3. 生命周期假设

- 身份契约：明确非人身份可能降低“欺骗人类感”和现实替代风险。
- 首次价值：视觉角色、语音与共同探索可能比纯文字更早建立对象感。
- 连续性证明：官方强调记忆，但写入、召回、纠错和删除粒度尚未实测。
- 退出：账号删除入口已由 FAQ 确认，情感退出措辞与回归体验未知。

## 4. 能力边界

当前没有 Web 产品入口，完整 onboarding、麦克风/相机权限、首个共同任务、主动消息和付费墙需要 iOS/Android 实机。现阶段不把官网演示写成产品实测。

2026-07-17 复核时，美国区 Google Play 的既有 Package 链接返回 404（EVD-137）。目前不能判断是下架、地区限制还是 Package 变化，因此 Android 评论采集暂停，保留为待交叉核验的产品状态风险。

## 5. 下一步

1. 获取可用移动端测试环境并完成免费首日流程；
2. 分层采集近期商店评论，重点编码记忆、语音、重复内容、付费和依赖；
3. 核验价格、通知控制、记忆删除与长期离开后的回归体验。
