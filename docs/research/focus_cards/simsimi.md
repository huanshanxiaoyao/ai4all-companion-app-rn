# 重点产品轻量卡：SimSimi

> 产品ID：PRD-088  
> 状态：首轮标准卡完成  
> 最近核验：2026-07-17  
> 证据性质：Google Play、App Store、官方 Web 与当前公开评论；经典众包机器人和新角色平台的关系待账号内实测

## 1. 定位与规模

- 关系对象：经典形态是由大量用户共同教导的 Everyone's SimSimi，也支持由单一所有者管理的 Personal SimSimi；当前 Web 又扩展为可搜索、创建和聊天的多角色平台。
- 平台：iOS、Android、Web；Android Package 为 `com.ismaker.android.simsimi`，iOS App ID 为 `375239755`。
- 规模：Google Play 100M+ 下载、约 4.2 分和 4.16M 评论；美国区 App Store 约 3.96 分和 7.5K 评分。（EVD-191、EVD-192）
- 目标用户：寻求即时娱乐、随机对话、轻社交和低门槛角色聊天的大众用户。

## 2. 核心关系机制

| 维度 | 当前已知 | 证据 |
| --- | --- | --- |
| 经典对话 | 回答来自大量用户教导的问答对，更接近集体语言反射而非稳定人格 | EVD-191 |
| 个体角色 | Personal SimSimi 由所有者管理，可代表所有者与他人聊天或自动回复 | EVD-191 |
| 平台迁移 | 当前官网已提供搜索、创建、角色目录、经典 SimSimi、Spicy 版本和多类 UGC 角色 | EVD-193 |
| 关系连续性 | 官方未公开足够的长期记忆、可见回忆或跨角色用户模型；当前不能把高规模视为深关系证据 | 待实测 |
| 轻社交 | 评论者提到通过产品结识朋友，说明价值不只来自机器人回复 | RVW-061 |
| 安全 | 官方提供内容政策、举报和年龄限制，但评论仍报告用户教导的敌意、性化内容和过滤失效 | EVD-191、RVW-063 |

## 3. 商业化

广告 + 会员订阅 + SimSimi Points/Boost。美国区 App Store 当前公开月度订阅和约 1.99–99.99 美元的点数/高级档位；Web 以登录赠点和 Boost 入口承接高级聊天。评论明确把广告密度和去广告会员视为主要摩擦。（EVD-192、EVD-193、RVW-062）

## 4. 评论线索

- RVW-061：认可通过产品结识朋友，但服务器速度显著拖累体验。
- RVW-062：认为广告过多并希望增加语音消息；开发者回复称会员可去广告。
- RVW-063：报告用户教导内容出现敌意和异常文本，并认为脏话过滤无效。

这些评论只用于识别稳定性、广告、语音和内容治理问题，不代表 100M+ 用户的总体分布。

## 5. 初步判断

- 可借鉴：把“经典固定角色”和“用户可创建角色”分层，让低门槛闲聊与更深角色关系共存。
- 需要改造：任何众包或 UGC 语言供给都必须经过发布前治理、可追责来源、区域化安全评估和用户可见的举报修复闭环。
- 不建议直接采用：把随机、有趣、冒犯性的集体语言反射当作稳定陪伴人格；规模和幽默不能补偿关系连续性与安全的不确定。

## 6. 证据缺口

- 新 Web 角色与经典 Everyone's/Personal SimSimi 的账号、记忆和内容关系；
- Boost 的模型、单轮消耗、免费获取、有效期与移动端映射；
- 长期记忆、用户资料、角色背景和跨设备同步；
- 真人轻社交、个人机器人自动回复与 AI 角色聊天之间的数据流和边界；
- 不同语言和地区的内容审核、年龄限制、申诉与过滤有效性；
- 语音消息、主动触达和离开后回归机制。
