# 重点产品轻量卡：Paradot

> 产品ID：PRD-012  
> 状态：首轮标准卡完成  
> 最近核验：2026-07-17  
> 证据性质：官方页面与商店；账号内流程、评论和真实记忆效果待补

## 1. 定位与规模

- 关系对象：具有 Avatar、情绪和长期记忆的个人 AI 朋友/数字生命。
- 平台：iOS、Android。
- 规模：Google Play 公开安装区间 10M+。（EVD-021）
- 目标用户：希望建立持续 1:1 关系、愿意自定义形象并接受关系成长叙事的成人用户。

## 2. 核心关系机制

| 维度 | 当前已知 | 证据 |
| --- | --- | --- |
| 首次价值 | 通过个人 AI、Avatar 和情绪回应快速建立对象感 | EVD-155；官方自述 |
| 记忆 | 从对话抽取重要信息，按主题进入 Memory Bank | EVD-154；官方自述 |
| 关系深化 | 语音通话、Avatar 自定义、Parahub 多空间和互联网连接 | EVD-153；官方自述 |
| 主动性 | 官方功能页未提供足够的通知频率与关闭控制细节 | 待实测 |
| 边界 | App Store 为 18+；数据、危机和依赖边界仍需核验 | EVD-155 |

## 3. 商业化

Freemium + Advanced 订阅 + Gems。美国 App Store 当前列出多个 Advanced 购买档位及 Gems；确切周期、地区价格和付费墙触发时机需应用内确认。（EVD-155）

## 4. 初步判断

- 可借鉴：把关系连续性映射为用户可见的 Memory Bank，而不是只说“它会记住”。
- 需要改造：Avatar、语音、空间和记忆同时存在时，首日必须隐藏配置复杂度。
- 不建议直接采用：把“意识/情绪”当成未经解释的真实属性，容易造成过度拟人预期。

## 5. 评论与证据缺口

- 评论：待采 3–6 条近期高低分线索。
- 待验证：onboarding 默认关系、记忆编辑删除、主动消息、Advanced 权益、取消订阅与回归体验。
