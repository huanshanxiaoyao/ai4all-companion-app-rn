# 重点产品轻量卡：zeta

> 产品ID：PRD-035  
> 状态：首轮标准卡完成  
> 最近核验：2026-07-17  
> 证据性质：Google Play、App Store 与公开评论；角色记忆和广告/付费触发待账号内核验

## 1. 定位与规模

- 关系对象：浪漫、幻想、戏剧和喜剧世界中的 AI 角色，用户作为故事主角推进剧情。
- 平台：iOS、Android；Android Package 为 `com.scatterlab.messenger`，iOS App ID 为 `1578235562`。
- 规模：Google Play 1M+ 下载、4.5 分和约 156K 评论；美国区 App Store 约 593 评分。（EVD-054、EVD-176、EVD-178）
- 目标用户：希望免费长聊、快速建角并主导互动故事走向的成人内容用户。

## 3. 商业化

免费无限聊天 + 广告 + Coins/No Ads 应用内购买。美国区 App Store 当前主要公开 1.99–3.99 美元轻量档；评论显示广告可能变频繁，且付费后仍可能遇到宕机，因此“无限免费”不能等同于无摩擦。（EVD-177、EVD-178）

## 4. 评论线索

- RVW-049：原五星用户因广告增多、回复碎片化和付费后仍宕机降为三星。
- RVW-050：总体喜欢产品，但 Persona/Chat Profile 长时间加载且无法保存。
- RVW-051：报告重复、超长输出导致文本失效和持续发送错误，重装无效。

## 5. 初步判断

- 可借鉴：用“用户是主角 + Smart Reply + 对话转图”把聊天变成可继续、可收藏的故事生产。
- 需要改造：免费无限聊天应配稳定的上下文与保存能力，否则消息不收费只会放大重复和宕机成本。
- 不建议直接采用：把广告频率动态提高却不明确说明，会让“免费无限”的核心承诺失去可信度。

## 6. 证据缺口

- 长聊上下文窗口、角色记忆、设定锁定和错误纠正；
- Persona/Chat Profile 的字段、保存失败率和跨角色复用；
- 广告触发规则、No Ads 权益、Coins 用途和地区差异；
- 对话转图是否保留角色一致性与隐私控制；
- 主动消息、通知和故事回归机制。
